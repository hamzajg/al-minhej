# Projection Engine Overview

Version 1.0

---

Purpose

The Projection Engine transforms immutable Knowledge Artifacts into user experiences.

Knowledge never changes.

Only its presentation changes.

---

Why

One Knowledge Artifact may appear as

• Reading Page
• AI Context
• Timeline
• Knowledge Graph
• Search Result
• Mobile Card
• Printable Study Sheet
• Flash Card
• Learning Journey
• Audio Companion

The Projection Engine generates each representation from the same underlying source.

---

Benefits

Single Source of Truth

Consistent UX

Deterministic AI

Future-proof architecture

Separation between domain and presentation
