# Project Philosophy

AlMinhej is developed as an AI-native software project.

Specifications are the primary source of truth.

Code is an implementation of the specifications.

Every implementation must reference:

- Product Vision
- Domain Model
- UX Philosophy
- Architecture Decision Records

No implementation may contradict the specification.

Every architectural decision must be documented.

Every AI coding agent must read the specification before implementing features.

The specification repository evolves first.

The implementation repository follows.
