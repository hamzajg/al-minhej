# Domain Overview

Version: 1.0

---

# Purpose

This document defines the business domain of AlMinhej.

Every architectural decision must originate from this document.

---

# Core Domain

The core domain is

Knowledge.

NOT

Course

Lesson

Quiz

Chapter

Page

Those are merely presentations of knowledge.

---

# Philosophy

Knowledge exists independently of the user interface.

Books are one way to organize knowledge.

Learning Journeys are another.

AI explanations are another.

Everything ultimately references the same underlying knowledge.

---

# Core Business Question

Instead of asking

"What lesson is this?"

The platform asks

"What knowledge does this represent?"

---

# Knowledge First

Every feature must ultimately operate on Knowledge Resources.

Examples

Read

Search

Bookmark

Reflect

AI

Timeline

Vocabulary

Scholar

All consume the same Knowledge Resources.

---

# Domain Goals

Preserve authenticity.

Encourage discovery.

Reduce cognitive load.

Promote contextual understanding.

Support lifelong learning.
